🧑‍💻 Profile Card — Stage 1B (HNG Task)

📌 Description

This project is a testable and responsive Profile Card built as part of the HNG Frontend Stage 1B task.

It demonstrates:

- Semantic HTML structure
- Accessibility best practices
- Responsive design
- Dynamic time using JavaScript

---

🚀 Live Demo

👉 https://silly-selkie-068aec.netlify.app/

---

📂 GitHub Repository

👉 https://github.com/Shadrach-stack/profile-card

---

✨ Features

- Semantic HTML ("article", "section", "nav", "figure")
- Responsive layout (mobile, tablet, desktop)
- Accessible avatar image with "alt" text
- Keyboard navigable links with visible focus
- Dynamic epoch time (milliseconds) using "Date.now()"
- Required "data-testid" attributes for testing
